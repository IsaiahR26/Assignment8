# Assignment8-FrontEnd-
WEB APPLICAITON 

DESCRIPTION

The frond end of the application allows users to interact with the web browser and its contents. 
The front end works to display information and collect user input. 

The goal of the front end works as a user interface that connects to the back end application, allowing access to its database.

INSTALLATINON AND SETUP

Prerequisites:

- Visual Studio Code
- Node.js
- Git (bash)
- Google or any other Web browser

STEPS TO RUN THE FRONTEND

1. Click on your repository.
2. Access the clone link within GitHub where the Code (in green) dropdown is.
3. Find the SSH tab
4. You'll see a url for the clone link, next to it, you can copy that url link to the clipboard.
5. Open the repository folder in your visual studio code and open your terminal.
6. Once you're in the terminal, type in: git clone git@github.com:IsaiahR26/Assignment8.git


USAGE INSTRUCTIONS

Navigate the front end by using the nav links to go through each page. Buttons and other interactions like forms
allows the user to interact with the web application and collect their data. Mouse inpusts are also collected and 
read from the web application.



API INTEGRATION

After collecting information and user inputs, the web application connects to the backend with different GET, POST, and other types of requests. Javascript is also used to connect the Web application and the Api application.


CONTRIBUTION GUIDLINES

- Create a new branch from main

- Submit a pull request.

Within Visual Studio Code:

- git checkout BRANCHNAME

- git commit -m "Enter a description what you added to the code here"

To Merge to Main:

- git checkout main
